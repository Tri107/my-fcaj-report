Below is a list of AWS technology blogs that I and my team have researched and translated during my internship, aiming to share useful knowledge about cloud computing:

###  [Blog 1 - Building an AI Gateway for Amazon Bedrock using Amazon API Gateway](3.1-Blog1/)
This post introduces how to use Amazon API Gateway as an AI Gateway layer to govern, authenticate, manage quotas, and control access to Amazon Bedrock's foundation models at scale. This solution helps businesses easily manage the lifecycle of Generative AI applications, control costs, and enhance security.

###  [Blog 2 - Optimizing cloud operations with Kiro power for AWS DevOps Agent](3.2-Blog2/)
This article discusses how "Kiro power" combined with the AWS DevOps Agent eliminates constant context switching during incident response. Engineers can investigate, identify root causes, and create fixes directly within their IDE by intelligently connecting metrics, traces, and system configurations.

###  [Blog 3 - Amazon S3 Annotations: Attach rich, queryable context directly to your objects](3.3-Blog3/)
This blog introduces the Amazon S3 Annotations feature, which allows users to attach rich, queryable context directly to objects stored in S3. This feature enhances the ability to organize, manage, and analyze data more effectively without having to alter the original data structure.
